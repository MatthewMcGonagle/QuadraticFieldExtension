Quadraticfieldextension

C++ classes for dealing with quadratic field extensions of the rational numbers.

View on GitHub Download .zip Download .tar.gz

Purpose

The purpose of this project is to create a C++ class library for dealing with a succession of quadratic field extensions of the Rational Numbers, Q. That is, the library is concerned with field extensions of Q by successive additions of square roots.
